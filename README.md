# Escola da Costela - Site institucional

## Integrantes da dupla
- Caio Neiva Santana
- Henry Andrade de Rezende

## Descrição do caso escolhido
O caso escolhido foi o restaurante Escola da Costela, localizado no Park Way, em Brasília-DF. A proposta do projeto é criar uma página institucional simples para apresentar o restaurante, seus pratos principais, informações de contato, horário, localização e orientações para encomendas.

## Necessidade identificada
A necessidade identificada foi organizar em uma página única as informações mais importantes para clientes que desejam conhecer o restaurante, confirmar horários, ver opções de pratos e fazer encomendas ou retiradas.

## Público-alvo
O público-alvo é formado por moradores do Park Way e de outras regiões de Brasília que procuram churrasco, costela, carnes assadas e opções para almoço de fim de semana ou pedidos para retirada.

## Objetivo do site
O objetivo principal do site é facilitar o contato do visitante com o restaurante e orientar a solicitação de encomendas.

## Processo de desenvolvimento
A dupla organizou o conteúdo em seções principais: início, pratos, apresentação da casa, informações úteis, encomendas e rodapé. Depois, foram definidos os estilos visuais, a responsividade, a interatividade com JavaScript e o uso introdutório de Bootstrap.

## Principais decisões do projeto
Foram utilizados elementos semânticos como header, nav, main, section, article, figure e footer. A paleta de cores foi baseada em tons terrosos e quentes, relacionados a churrasco, madeira, brasa e comida caseira. A responsividade foi aplicada com Flexbox, Grid, Bootstrap e media queries.

## Funcionalidade em JavaScript
O projeto possui duas interações principais. A primeira mostra e oculta orientações de encomenda ao clicar no botão "Solicitar encomenda". A segunda altera uma mensagem de sugestão quando o usuário clica em um card de prato.

## Uso de Bootstrap
O Bootstrap foi utilizado de forma introdutória no grid da seção de informações úteis, com classes como row, col-md-6, col-lg-3, g-4 e h-100. Também foi usado de maneira leve nos botões, sem substituir o CSS próprio do projeto.

## Testes realizados
Foram testados os links internos, as imagens, a responsividade em diferentes larguras de tela, a navegação por teclado nos cards interativos, o funcionamento do JavaScript e a organização dos arquivos.
